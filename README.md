# Eleventastic

A simple Eleventy Starter Kit, my base for all new 11ty projects.

## Features

* CSS Pipeline (Sass, CleanCSS)
* JS Bundling (Webpack)
* SVG Icon Sprite Generation
* Critical CSS
* HTML Minification
* Hot Reloading
* No external builds, everything runs through 11ty
